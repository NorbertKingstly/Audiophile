Audiophile – Project Walkthrough

Project Overview
Audiophile is a modern music streaming web application inspired by Spotify and Apple Music. The project is built using HTML, CSS, and JavaScript with a clean UI, smooth music playback, album browsing, artist pages, favorites system, profile management, and customizable themes.

The project includes:
* Login & Register system
* Dynamic Home Page with carousel banners
* Songs Library with music player
* Albums Page
* Artists Page
* Favorites System
* User Profile & Settings
* Privacy Policy & Terms Pages
* Bug Reporting Page
* Cloud-based audio streaming using Cloudinary URLs

1. Authentication System
When the user opens Audiophile, they first enter the authentication section where they can create an account or log in securely.

Features:
* Modern glassmorphism UI
* Responsive authentication forms
* Login & Register pages
* Forgot Password page
* Stylish gradients and blur effects

Files:
* index.html
* register.html
* forgot.html

2. Home Page
After login, users enter the main home dashboard where trending music, banners, and recommendations are displayed.

Features:

3. Songs Module
